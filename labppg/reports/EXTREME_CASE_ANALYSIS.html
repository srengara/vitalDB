3. Cardiovascular Confounding Evidence

Core Finding: Blood Pressure and Heart Rate Drive Prediction Error

The model was trained on VitalDB surgical ICU data where hemodynamic instability (high HR, high BP, stiff vasculature) co-occurs with hyperglycemia. When applied to ambulatory subjects, it mistakes cardiovascular-driven PPG features for glucose-driven optical absorption changes.

7. Root Cause Analysis

Finding 1: Cardiovascular State is the Dominant Confound

  • Systolic BP correlates r = +0.301 with absolute error — the strongest predictor
  • Actual glucose level correlates r = +0.020 with error — essentially zero
  • The 3 worst-performing persons (Mks, Vikram, Vikraman) all have elevated HR (>100) or elevated BP (>140 systolic) or both
  • The 3 best-performing persons (Harsha, Anindita, Sudha) have normal cardiovascular profiles (HR < 85, BP < 135)

Finding 2: ICU Training Data Creates Spurious Correlations

  • The model was trained on VitalDB surgical ICU data where hemodynamic instability co-occurs with hyperglycemia
  • Elevated BP → increased vascular stiffness → stiffer pulse waveform → earlier, sharper reflected wave
  • Elevated HR → shorter diastole → compressed dicrotic notch → altered PPG morphology
  • The model learned: "stiff pulse waveform = high glucose" from ICU patients, but this association breaks for ambulatory subjects

Finding 3: Signal Quality Metrics Do NOT Predict Accuracy

  • The worst case has better signal quality: 25 accepted windows, similarity = 0.967
  • The best case has worse signal quality: only 2 accepted windows, similarity = 0.743
  • High window consistency means the PPG waveform is reproducible — not that the model will interpret it correctly
  • The model is confidently and consistently wrong for Vikraman: all 25 windows predict 172–287 mg/dL when actual is 102

Finding 4: The "Perfect" Prediction is Statistical Coincidence

  • Pramod's 0.1 mg/dL error comes from only 2 surviving windows that happened to average near truth
  • The same person's other 13 readings have MAE ranging from 0.1 to 75.6 mg/dL (mean: 32.3)
  • The same person, same timestamp, on IR PPG gives ~50 mg/dL error
  • This is not genuine model accuracy but rather the law of small numbers

8. Recommendations

For Model Improvement

  1. Cardiovascular Normalization: Add HR and BP as auxiliary inputs or condition the model on hemodynamic state to disentangle vascular from metabolic PPG features
  2. Domain Adaptation: Fine-tune with ambulatory PPG data rather than relying solely on ICU data, which has different hemodynamic profiles
  3. Multi-Task Learning: Jointly predict glucose + HR + BP so the model learns to separate these effects
  4. Prediction Confidence: Use prediction standard deviation as a confidence metric — flag cases where pred_std > 20 as unreliable (r = +0.359 correlation with error)
  5. Subject-Specific Calibration: A per-subject calibration step using 1–2 reference glucose readings could dramatically reduce person-level bias

For Data Collection

  1. Record BP and HR alongside PPG to enable post-hoc cardiovascular normalization
  2. Collect from diverse cardiovascular profiles — current data is biased toward normotensive subjects
  3. Include glucose challenge tests (oral glucose tolerance) where the same person has readings at 80, 120, 180+ mg/dL to separate person-level bias from glucose sensitivity
  4. Multiple readings per session to estimate within-session variability and identify unreliable recordings
